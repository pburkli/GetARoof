# Stripe — Notes

## VCC Issuing — Switzerland

Stripe does not issue virtual cards (VCC) for businesses or individuals located in Switzerland. Stripe Issuing is not available in CH.

**Stripe is not a viable option for the GetARoof payment + VCC workflow when operating from Switzerland.**

See `platforms/airwallex/api.md` for the current VCC issuing approach (note: Airwallex CH availability also needs verification).
